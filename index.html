Group Anagrams

The "Group Anagrams" problem, a popular challenge in algorithmic interviews and on platforms like LeetCode, requires grouping words that are anagrams of each other. An anagram is a word formed by rearranging the letters of another word, using all original letters exactly once.

By removing spaces and converting the strings to lowercase letters, comparing the lengths, and sorting the characters in both strings, we are able to check if the two strings are anagrams of each other or not.EX: if the array is {"eat", "bat", "ate"}, the output of the problem will be [["bat"], ["ate", "eat"]].

Given an array of strings strs, group the anagrams together. You can return the answer in any order This code uses a Map to store anagrams. It iterates through each string in the input array, sorts its characters, and uses the sorted string as a key in the map. If the key is not present, it creates a new array for that key; otherwise, it adds the current string to the existing array. Finally, it converts the map values to an array and returns the result
